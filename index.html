prolog

@adrianomelo

prolog data objects (termos)

estrutura

Elemento que representa uma coleção de termos

É constituida por um functor (símbolo constante) e um ou mais componentes

unificação de estruturas

duas estruturas unificam se:

1. functor é o mesmo

2. possuem a mesma aridade

3. todos os componentes unificam

O símbolo "_" é unificado com qualquer valor

A unificação de estruturas aninhadas é feita recursivamente

1. O functor e a aridade são os mesmos

2. Os componentes unificam

?- f(a, b, Z) = f(X, Y, c).
Z = c,
X = a,
Y = b.

?- X = f(X).
X = f(**).

?- X = f(Y), Y = g(X).
X = f(g(**)),
Y = g(f(**)).

?- X = s(X), Y = y(X,X), A = s(s(s(A))), B = y(A,A), Y = B.
X = s(**),
Y = y(s(**), s(**)),
A = s(s(s(**))),
B = y(s(s(s(**))), s(s(s(**)))).

unificação de listas

append(?List1, ?List2, ?List3)

Succeeds when List3 unifies with the concatenation of List1 and
List2. The predicate can be used with any instantiation pattern
(even three variables).

?- append(A, B, []).
A = [],
B = [] ;
false.

?- append([], B, [a]).
B = [a].

?- append(A, B, [a,b]).
A = [],
B = [a, b] ;
A = [a],
B = [b] ;
A = [a, b],
B = [] ;
false.

?- append(A, B, [X,b]).
A = [],
B = [X, b] ;
A = [X],
B = [b] ;
A = [X, b],
B = [] ;
false.

?- append([a], B, [X, f(X)]).
B = [f(a)],
X = a.

?- append(B, [X], [g(_)]).
B = [],
X = g(_G239) ;
false.

?- append(B, [g(_)], [g(_), g(a)]).
B = [g(_G241)] ;
false.

?- append(B, [X], [f(X)]).
B = [],
X = f(**) ;
false.
